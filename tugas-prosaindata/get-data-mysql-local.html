Cara menghubungkan MySql Local dengan Google Colaboratory#

  1. Langkah pertama, Kita memerlukan akses untuk terhubung dengan komputer lokal kita, caranya adalah kita harus memiliki token terlebih dahulu agar bisa connect

  2. Token kita dapatkan ketika kita pertama kali membuka Jupyter notebook anaconda

  3. Kita salin token tersebut dan tempelkan di Google Colaboratory kita

  4. Dengan cara sambungkan ke runtime lokal

  1. Dan mungkin anda akan menemui error seperti ini

Yang berarti anda harus mengupdate jupyter notebook di komputer anda. dengan cara membuka, Anaconda prompt dengan run as administator

  1. Kemudian kita ketik 2 perintah yaitu :

pip install upgrade jupyter_http_over_ws>=0.0.7 && jupyter serverextension enable py jupyter_http_over_ws

jupyter notebook NotebookApp.allow_origin=https://colab.research.google.com port=8888 NotebookApp.port_retries=0

  1. Jika sudah anda akan mendapatkan token dan kemudian salin, token tersebut, Jika langkah 6 anda lakukan dengan benar maka, Google colab anda akan terhubung ke server lokal komputer anda

pip install mysql-connector-python
